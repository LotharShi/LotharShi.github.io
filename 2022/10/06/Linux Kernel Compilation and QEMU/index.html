QEMU

What is QEMU?

QEMU is an open source machine emulator and virtualizer. I recognize it as a virtual machine software, like VMWare or VirtualBox. But its advantage is more about hardware virtualizaion.

It works on the user layer of the local machine, which indicates that plenty of the privileged hardware and kernel instrucitons are not available. So it may perform poorly, and normally will cooperate with KVM to realize the virtualization.

Install QEMU

There are two ways, with apt install or compile it yourself.

The first method is preffered since it’s easier to implement and time saving.

(After I tried both ways, I feel like nobody is gonna try the latter method unless he/she is boring enough and have plenty of time and space…)
